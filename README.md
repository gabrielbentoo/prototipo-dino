
# Prototipo


prototipo do jogo t-rex running


## Aprendizados

Criar prototipos para desenvolvimento de jogos


## Funcionalidades

- pulo com a tecla espaco do teclado para pular os obstaculos



## Stack utilizada

**Front-end:** html, css e javaScript




## Deploy

1. Clone ou baixe este repositório.
```bash
  git clone https://github.com/gabrielbentoo/prototipo-dino
```
2. Abra o arquivo index.html em seu navegador.
